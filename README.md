# 企业资产管理系统 java56

## 项目概述

企业资产管理系统是基于Springboot+Vue的前后端分离项目。采用经典的MVC三层架构，结合Spring、SpringBoot、Mybatis和Vue.js等技术进行实现。系统使用MySQL数据库作为数据存储，提供一个基于B/S结构的Web操作界面，界面简洁，操作方便。

## 技术栈

- 后端：Javaee、Springboot、Mybatis
- 前端：Vue.js
- 构建工具：Maven
- 数据库：MySQL

## 功能模块

- 资产信息管理
- 资产分类管理
- 资产使用状态跟踪
- 资产维护记录
- 用户权限管理

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- JDK 1.8 或以上版本
- MySQL 5.7 或以上版本
- Node.js 12.0 或以上版本
- Maven 3.6 或以上版本

## 部署步骤

1. 准备开发环境，安装上述指定的JDK、MySQL、Node.js和Maven版本。
2. 创建数据库，导入项目提供的SQL文件。
3. 使用Maven构建后端项目，并启动后端服务。
4. 在前端项目目录下运行npm install安装依赖。
5. 运行npm run dev启动前端开发服务器。
6. 访问前端界面进行操作。
